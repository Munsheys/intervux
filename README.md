# 🎙️ Intervux - AI Mock Interview

> Master your next interview with real-time AI roleplay and instant STAR-method feedback.

Intervux is a premium, browser-based mock interview platform designed to simulate the pressure of a real technical or behavioral interview. By utilizing the Web Speech API and tailored LLM prompts, it provides a low-latency, conversational experience that helps candidates refine their communication skills and master the STAR method.

---

## ✨ Features

- **🎯 Context-Aware Interviewing**: Paste your target Job Description, and Intervux generates tailored behavioral questions specific to that role.
- **🗣️ Voice-Activated Roleplay**: Speak naturally. The AI uses the Web Speech API to transcribe your answers in real-time, mimicking a real conversation.
- **📊 Real-Time STAR Feedback**: Instant analysis of your responses. The UI visually tracks if you've covered the **Situation, Task, Action, and Result**.
- **📹 Video Simulation**: The interface mimics a real video call, keeping your webcam active to help you maintain eye contact and professional presence.
- **🌑 Premium Design**: A stunning dark-mode glassmorphic UI designed for focus and clarity.

---

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (ES6+)
- **Speech**: Web Speech API for low-latency transcription
- **Design**: Responsive Grid/Flexbox with glassmorphic aesthetics
- **Icons**: FontAwesome 6

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Edge, or Safari recommended for Web Speech API support).
- Microphone and Webcam access.

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Munsheys/intervux.git
   ```
2. Open `index.html` in your browser.

### Usage
1. Enter the details of the job you are applying for.
2. Grant permission for Microphone and Camera access.
3. Start the interview and answer the questions naturally.
4. Review your performance data after each session.
