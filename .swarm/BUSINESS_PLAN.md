## 🏰 Moat
*Written by VC-001*

Intervux's moat relies on **UX and Real-Time Latency Optimization**. By running speech recognition locally via Web Speech API and streaming structured JSON from an LLM for the feedback, we create an ultra-low latency loop that feels like a real interview. Competitors relying on full server-side voice agents (like Vapi) suffer from awkward pauses that break the simulation. Our "Dashboard" UI makes the feedback immediately actionable.
